Autonomous Parking Agent (Engineering Physics Project Course, UBC)

Nov - Dec 2022

In this course, my partner and I developed an autonomous agent that drives through a ROS environment while obeying traffic laws. Real-time images from the in-simulation camera view from the robot are the only input received from the environment. There are two different ROS nodes that process the images: one for publishing velocity commands to steer the robot and another to publish the license plate numbers to a license plate tracker. The nodes run simultaneously to drive the robot while detecting license plates. My main focus in the project was on successfully detecing license plate numbers.

The flowchart above demonstrates how we controlled the autonomous agent.

An example of how we used contour detection to find parking and plate numbers is shown above.

We used an HSV filter function to better detect the characters from the license plate. We proceeded to train a convolution neural network for character recognition.
